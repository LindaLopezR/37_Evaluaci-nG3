5) Menciona las estucturas condicionales y menciona un ejemplo donde se utilice cada una:

IF - Cuando se tiene una condición si es cierta se ejecuta la acción y termina el programa, si es falsa termina el programa.

IF ELSE - Cuando se tiene una condición al ser cierta se ejecuta una acción determinada y termina el programa, si es falsa realiza otra acción determinada y termina el programa

IF ELSEIF - Se utiliza cuando se tiene diferentes condiciones, cuando la condicion no abarca el problema, es decir cuando se cumpla dicha condición hacer esto, sino si se cumple otra condición has esto sino lo otro, etc.

CASE - Se utiliza cuando no se tiene parámetros sino respuestas concretas y fijas de la condición
